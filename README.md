About<br>
============================================
这个高精计算计算被我用来作为C语言学习的练习项目，所以只在力所能及的范围内考虑代码的运算效率。<br>

通过函数原型来创建每个运算符，并最终计划用一个统一的函数原型来调度，做到简化完成后用来运算所需要的代码量。<br>
